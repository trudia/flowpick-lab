# TrendFlow V79 업데이트

## 변경 사항

- 홈 `최근 글` 영역을 이미지/설명 카드형에서 제목만 보이는 텍스트 리스트형으로 변경했습니다.
- 홈 `추천 시리즈` 영역을 2×2 팜플렛형 카드 UI로 확대했습니다.
- 추천 시리즈 내부 글은 이미지 + 짧은 제목으로 유지하되, 썸네일을 다시 크게 조정했습니다.
- V78의 강제 축소 CSS가 남아 있어도 V79 스타일이 우선 적용되도록 `!important` 오버라이드를 추가했습니다.
- `styles.css?v=79`, `theme.js?v=79` 캐시 버스터를 반영했습니다.
- 배포 확인용 `/deploy-check-v79.html` 파일을 추가했습니다.

## 업로드 방법

ZIP 압축을 풀고 GitHub 저장소 루트에 전체 파일을 그대로 덮어쓰기 업로드하세요.
업로드 후 `https://trend.it.kr/deploy-check-v79.html` 이 보이면 V79 파일이 반영된 것입니다.
