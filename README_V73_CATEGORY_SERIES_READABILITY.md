# TrendFlow V73 - Category Series + Korean Readability Update

이 업데이트는 V72 라이트/다크 모드 구조를 유지하면서 홈/카테고리 추천 시리즈와 한국어 줄바꿈 가독성을 보완합니다.

## 변경 사항

- 홈 추천 시리즈를 단일 시리즈 교체 방식에서 `카테고리별 추천 시리즈` 누적 방식으로 변경
- 기존 `AI 생산성 입문 시리즈` 삭제하지 않고 유지
- `집중력 회복 시리즈`, `공간 자극 줄이기 시리즈`, `구매 기준 정리 시리즈`를 함께 노출
- 각 카테고리 페이지에 `읽는 순서` 시리즈 박스 추가
- 한국어 문장이 카드/본문에서 어색하게 줄바꿈되는 문제 완화
- `word-break: keep-all`, `text-wrap: pretty/balance`, `overflow-wrap: break-word` 적용
- V72 다크/라이트 모드 유지
