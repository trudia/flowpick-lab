# TrendFlow 쇼핑글 링크 요청 템플릿

아래 형식으로 사용자에게 쿠팡파트너스 링크를 요청합니다.

```text
TrendFlow 쇼핑글 후보 분석 결과입니다.

주제:
[주제]

메인 키워드:
[메인 키워드]

검색 의도:
[방문자가 구매 전에 확인하려는 기준]

글 유형:
[문제 해결형 / 비교형 / 실패 방지형 / 시즌형 / 세트형]

글 구성 기준:
- 2,500~3,500자 내외
- 파소나 흐름 기반
- 제품을 단순 추천하지 않고 사용자 상황, 가성비, 장단점, 제외 기준을 함께 설명
- 글에 맞는 대표 이미지 사용
- 쇼츠/릴스 전환 가능한 훅 1개 포함

필요한 쿠팡파트너스 링크:
1. [상황 / 제품 유형 1]
2. [상황 / 제품 유형 2]
3. [상황 / 제품 유형 3]
4. [상황 / 제품 유형 4]
5. [상황 / 제품 유형 5]

위 링크를 주시면 기존 trend-shopping UI에 맞춰 글, 제품 카드, CTA, 홈/카테고리/검색/아카이브/사이트맵 반영까지 진행하겠습니다.
```

사용자가 링크를 제공할 때 기대하는 형식:

```text
주제:
[주제]

쿠팡 링크:
1. [제품 유형] - https://link.coupang.com/...
2. [제품 유형] - https://link.coupang.com/...
3. [제품 유형] - https://link.coupang.com/...
4. [제품 유형] - https://link.coupang.com/...
5. [제품 유형] - https://link.coupang.com/...
```
