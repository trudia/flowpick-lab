# TrendFlow Daily Blog Operation Brief

## 오늘 판단할 것

1. 오늘은 신규 글, 기존 글 보강, 쇼핑 글, 내부 링크 개선 중 무엇이 가장 효과적인가?
2. 네 가지 핵심 영역 중 어느 영역이 부족한가?
   - AI 입문
   - 집중 회복
   - 공간 정리
   - 구매 기준
3. 쇼핑 의도가 있는 주제인가?
4. 쿠팡파트너스 링크가 필요한가?
5. 링크가 없다면 사용자에게 어떤 제품군 링크를 요청해야 하는가?

## 출력 형식

```text
오늘의 운영 제안:
[신규 글 / 기존 글 보강 / 쇼핑 글 / 내부 링크 개선 / 별도 제안 없음]

이유:
[짧은 이유]

진행할 주제:
[주제]

필요한 사용자 입력:
[없음 / 쿠팡파트너스 링크 3~5개 / 승인 필요]

[Codex가 바로 진행할 작업]
```
