# V156 Image Sourcing Policy

Date: 2026-06-04

## Summary

- Updated the TrendFlow operation process so article images do not have to be AI-generated.
- Licensed stock images, including Getty Images Bank-style sources, may be used when they fit the site and the license permits publication and editing.
- Image edits such as cropping, color adjustment, compression, and composition cleanup are allowed when they improve article fit.
- External stock image use should record source, license basis, download date, and edits in a README or internal audit note.
- Product detail page images, review images, unlicensed search-result images, and screenshots from other publishers remain disallowed.

## Practical Rule

Use the image that best helps the visitor understand the article:

1. Licensed stock image when a real-world scene is more convincing.
2. New AI-generated image when no suitable licensed visual exists.
3. Existing site image only when it clearly matches the article and is not reused across unrelated topics.
