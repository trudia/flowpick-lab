# TrendFlow V75 - 시리즈 추천 이미지 카드형 UI 업데이트

## 반영 내용

- 홈 추천 시리즈 영역에서 `카테고리 보기` 버튼 제거
- 상단 내비게이션에 이미 카테고리가 있으므로 시리즈 카드 내부 링크 버튼을 정리
- 1단계~5단계 설명형 목록을 `이미지 + 짧은 제목` 카드형 UI로 변경
- 홈 추천 시리즈와 각 카테고리 페이지의 읽는 순서 영역 모두 반영
- 긴 설명문 대신 요약 제목 중심으로 표시
- 기존 V72 라이트/다크 모드 유지
- 기존 V73 한국어 줄바꿈 보정 유지
- 기존 V74 구분감 있는 카드/섹션 디자인 유지
