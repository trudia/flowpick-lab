# V104 Subscribe & Privacy Basic Patch

반영 내용:
- 글 상세페이지 구독 박스를 `구매 체크리스트 받기` 구조로 변경
- Google Form 연결용 placeholder 링크 삽입
- 개인정보처리방침 페이지 추가: `/privacy.html`, `/privacy/`
- 구독 박스에 개인정보처리방침 링크 삽입
- 다크모드 구독 박스/개인정보처리방침 가독성 스타일 포함
- footer의 개인정보처리방침 링크 유지

구글폼 연결 방법:
1. Google Forms에서 구독 신청 폼을 생성합니다.
2. 공유 링크를 복사합니다.
3. 상세 글 HTML에서 `https://forms.gle/REPLACE_WITH_YOUR_GOOGLE_FORM_URL` 값을 실제 구글폼 링크로 교체합니다.
4. 구글폼 첫 문항 또는 설명에 개인정보 수집·이용 동의 문구를 넣습니다.
