# TrendFlow V76 Compact Series Thumbnails Update

## 변경 목적
V75에서 추천 시리즈 영역의 이미지가 너무 커 보여 홈 UI의 균형이 무너지는 문제를 수정했습니다.

## 반영 내용
- 추천 시리즈 카드 크기를 `주제별로 보기` 카드와 비슷한 밀도로 축소
- 추천 시리즈 내부 이미지를 작은 썸네일 스트립 형태로 변경
- 카테고리 페이지의 `읽는 순서` 카드도 작은 이미지 + 제목 중심으로 축소
- 긴 설명은 유지하되 시각적으로 덜 튀도록 숨김/축소 처리
- 기존 추천 시리즈와 글 목록은 삭제하지 않고 유지
- V72 라이트/다크 모드, V73 한국어 줄바꿈 보정, V74 구분선, V75 이미지+제목 구조 유지
