제휴 안내

트렌드플로우는 상품을 먼저 추천하지 않습니다. 먼저 문제 상황을 확인하고, 구매 전 판단 기준을 본 뒤, 필요한 경우에만 제휴 링크를 확인하세요.

운영 기준

  • 상품보다 문제 해결 기준을 먼저 확인하도록 안내하세요.
  • 사지 않아도 되는 경우와 실패 패턴도 함께 보여주세요.
  • 단일 상품 강매보다 상황별 조합과 비교 기준을 제공합니다.
  • 실제 쿠팡파트너스 링크는 발급 후 `data/affiliate-links-template.json` 또는 각 글의 `수정예정` 위치를 교체하면 됩니다.

링크 교체 방법

  1. 쿠팡파트너스에서 상품 링크를 발급합니다.
  2. 글 안의 `쿠팡 링크 등록 예정` 버튼 주소를 실제 링크로 바꿉니다.
  3. 링크에는 가능한 `rel="nofollow sponsored noopener"` 속성을 유지합니다.
  4. 제휴 고지 문구를 삭제하지 않습니다.

운영자 편집 가이드

글과 이미지를 직접 수정하려면 운영자 편집 가이드와 글 HTML 생성기를 사용할 수 있습니다.

운영용 데이터 파일

data/affiliate-products.json에 카테고리별 상품군, 우선순위, 링크 교체 위치를 정리했습니다.
