# 이미지 업로드 위치

V37은 자동 생성 이미지 대신 사용자가 제공한 이미지를 넣기 위한 구조입니다.

## 카테고리 이미지
assets/uploads/categories/
- ai-productivity.webp
- focus-wellbeing.webp
- minimal-space.webp

## 글 이미지
assets/uploads/posts/<post-slug>/cover.webp

예:
assets/uploads/posts/sidehustle-gear-priority-v18/cover.webp

## 권장
- 1600x1000 또는 1920x1200
- WebP 또는 JPG
- 비교형 글: 좌우 비교 이미지
- 전후형 글: before / after 이미지
