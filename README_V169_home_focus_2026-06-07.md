# TrendFlow v169 Home Focus Update

## Summary

- Simplified the homepage so first-time visitors see fewer choices.
- Removed repeated homepage routes and long series blocks from the first page.
- Kept the homepage focused on one promise: visitors should leave with one useful standard.

## Homepage Structure

- Hero slider with a short brand promise.
- "Today one thing" quick panel.
- Three grouped entry routes:
  - Rainy season room humidity
  - AI and career judgment
  - Side hustle and buying criteria
- Compact category pills.
- Recent posts list.

## Reason

The previous homepage exposed too many sections at once. This update makes TrendFlow feel more like a guided curation page than a dense blog index.
