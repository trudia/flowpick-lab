# TrendFlow V57 Blog Reset

복잡한 플랫폼형 구조를 중단하고 블로그형 글쓰기 사이트로 리셋한 버전입니다.

## 운영 방식

- 글 원본: `_posts/*.md`
- 정적 글 페이지: `posts/<slug>/index.html`
- 카테고리: `categories/<slug>/index.html`
- 검색: `search/index.html`
- 글쓰기 보조: `editor.html`

## GitHub는 고정인가?

필수는 아니지만 현재 trend.it.kr이 GitHub Pages에 연결되어 있다면 그대로 쓰는 것이 가장 쉽습니다. 다른 호스팅으로 옮길 수도 있지만, 지금은 기존 GitHub 저장소를 유지하고 파일 구조만 단순화하는 것을 권장합니다.
