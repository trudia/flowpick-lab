# TrendFlow V72 Theme Mode Update

이번 업데이트는 V71 전체 사이트 구조를 유지하면서 라이트/다크 모드 기능을 추가한 GitHub 업로드용 전체 사이트 ZIP입니다.

## 변경 사항

- 사이트 전체 라이트/다크 모드 토글 추가
- 사용자가 선택한 테마를 `localStorage`에 저장
- 첫 방문자는 OS 기본 설정(`prefers-color-scheme`)을 따라 자동 적용
- 라이트 모드 배경을 순백색보다 부드러운 블루-그레이 계열로 조정
- 다크 모드에서 배경, 카드, 본문, 링크, 버튼, 입력창, 사이드바 색상 자동 전환
- 모바일 화면에서는 토글 버튼을 아이콘 중심으로 축소
- 기존 V71 글, 카테고리, 홈, 검색, 아카이브, 사이트맵 구조 유지

## 업로드 방법

1. ZIP 압축을 풉니다.
2. GitHub 저장소 루트에 전체 파일을 그대로 덮어씁니다.
3. GitHub Pages 배포가 완료된 뒤 새로고침합니다.
4. 상단 우측 `Dark / Light` 버튼으로 테마를 전환합니다.

## 주요 파일

- `styles.css` : V72 테마 색상 및 다크모드 스타일 추가
- `assets/theme.js` : 테마 저장/전환 스크립트
- `index.html` 및 모든 하위 HTML : 테마 토글 버튼과 초기 테마 적용 스크립트 반영
