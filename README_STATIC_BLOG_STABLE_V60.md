# TrendFlow V60 Static Blog Stable

V59 Jekyll 방식이 GitHub Pages에서 바로 반영되지 않는 문제를 피하기 위해,
이번 V60은 **Jekyll 빌드 없이 바로 보이는 정적 블로그 버전**입니다.

## 핵심

- `.nojekyll` 포함
- 홈, 카테고리, 검색, 글 상세 HTML을 모두 정적으로 생성
- `_posts/*.md`는 글 원본 보관용으로 유지
- GitHub Pages 빌드 실패와 관계없이 바로 서비스 가능

## 이번 추가 글

- 반복 작업을 줄이는 프롬프트 라이브러리 구축법
- 메모를 글감으로 바꾸는 AI 아이디어 정리 루틴
