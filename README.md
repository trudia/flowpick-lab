# 트렌드잇랩 polished site pack

화면 문구와 UI를 다듬은 버전입니다.

## 변경 요약
- 방문자에게 보이는 `MVP`, `예시`, `준비 중`, `링크 준비 중` 문구 제거
- 메인 제목과 섹션 제목을 더 자연스럽게 수정
- 히어로 글자 크기와 카드 간격 조정
- 이메일 입력 영역을 빠른 가이드 링크 영역으로 변경
- 상품 버튼을 실제 카테고리 페이지로 이동하도록 연결
