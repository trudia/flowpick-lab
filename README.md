# 트렌드잇랩 MVP 콘텐츠 확장팩

포함 파일:
- index.html
- styles.css
- posts/ 글 3개
- categories/ 카테고리 4개
- robots.txt
- sitemap.xml

업로드 방법:
1. 압축 해제
2. GitHub 저장소에서 Add file → Upload files
3. 모든 파일과 폴더를 그대로 업로드
4. 기존 index.html, styles.css는 덮어쓰기
5. Commit changes
6. Cloudflare Pages 자동 배포 확인

쿠팡파트너스 링크는 각 글의 `링크 준비 중` 부분을 교체하면 됩니다.
