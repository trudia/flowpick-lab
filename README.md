# 트렌드플로우 사이트 최신화 v3

확인된 문제:
- 홈페이지가 이전 버전으로 남아 사이트명이 트렌드잇랩으로 표시됨
- 홈페이지에 MVP, 고지 예시, 실제 배포 시 등 운영자용 문구가 남아 있음
- 글/약관 페이지와 홈페이지의 톤이 달라 보임

반영 내용:
- 홈페이지까지 트렌드플로우 기준으로 통일
- 운영자용 문구 제거
- 홈 화면 우측 카드를 방문자용 가치 설명으로 변경
- 이메일 입력 영역 제거 후 빠른 가이드 링크 영역으로 변경
- 푸터 개인정보처리방침 강조 유지
- privacy.html, terms.html, 404.html 포함
- UI 간격, 버튼, 카드, 모바일 표시 개선

업로드 방법:
1. 압축 해제
2. GitHub 저장소에서 Add file → Upload files
3. 모든 파일과 폴더 업로드
4. 기존 파일은 덮어쓰기
5. Commit changes
6. Cloudflare Pages 배포 확인
